polynomial time

暂无

## 中文释义

问题需要的时间（复杂度）与问题的规模之间是多项式关系。

多项式关系形如$O(kn)O(n^k)$，k为某个常数，n是问题的输入规模。

在计算机学科中，存在多项式时间的算法的一类问题，称之为P类问题。
