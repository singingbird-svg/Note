## 中文释义

将一个复杂的问题或算法分解成很多个小的单元，每个单元称为一个模块 (module)。这些模块会被进一步分解成更小更基本的单元，直到这些小单元已经足够简单、很容易被理解，或者可以转化为已经解决的问题，此时即可以停止进一步分解。
