## 英文
 
Automated algorithm selection

## 英文释义

to automatically select from a portfolio of chosen algorithms with their associated parameters upon a set of training problem instances

## 中文释义

在一组训练问题实例上，从所选算法及其相关参数的组合中自动选择。
